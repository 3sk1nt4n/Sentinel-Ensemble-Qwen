🛡️ Sentinel Ensemble — How It Works

CONDUCTOR · deterministic Python
owns every step — the AI is called 5 times (✦), and never touches a shell
✦ AI CALL the 5 AI invocations ⭐ SELF-CORRECTION AI revises its own verdicts (twice) ⚙ code deterministic — checks the AI
0–4
SETUP
⚙ Onboarding & Setup
Mount evidence read-only, take a SHA-256 fingerprint, check the kernel for rootkits, and build the catalog of 195 forensic tools — no tools run yet.
▼
5
✦ AI
AI Call 1 · Tool Selection
The AI picks the tools — from 190+ typed tools on the MCP server, never a shell command — then deterministic code injects the must-have tools it can’t skip, so coverage is guaranteed.
▼
6–7
COLLECT
▦ Collect Evidence
Tools run as typed functions in parallel (never a shell); every output becomes typed facts + a paired reference set.
▼
8–9
✦ AI
AI Call 2 · Analysis⚡ 4-MODEL ENSEMBLE
Four AI models run in parallel (an ensemble): each analyzes the evidence on its own, and only what they agree on is kept — cross-validation built in.
▼
10
⚙ CODE
🧪 Validate
Code — not the AI — checks every claim → MATCH ✓ · MISMATCH ✗ (blocked) · UNRESOLVED. Runs in parallel with Step 11; blocked findings aren’t dropped — they go to Step 13AA.
▼
11
✦ AI
AI Call 3 · ReAct Cross-Check⭐ SELF-CORRECT #1
While the validator runs, the AI re-investigates its suspicious findings with live tools and overturns its own verdicts — clearing false alarms.
▼
12–13
CALIB
📊 Calibrate Confidence
Confidence is earned by independent evidence: 3+ types = HIGH · 2 = MEDIUM · 1 = LOW. Blocked findings are rescued, never dropped.
▼
13AA
✦ AI
AI Call 4 · Final Self-Correction⭐ SELF-CORRECT #2
The agent re-judges every still-uncertain finding (needs-review · inconclusive · blocked/rejected · low/info) — promote, keep, or clear. Code gates it: no promotion without evidence.
▼
14
✦ AI
AI Call 5 · Report
The AI writes the investigative narrative from the validated truth — and its words can never contradict it.
▼
15–16
DELIVER
🔒 Verify & Deliver
SHA-256 matches — evidence was never modified — then the report, findings table, IOCs, and interactive dashboard are delivered.
5 AI calls · checked by code at every turn · self-corrects twice · evidence is never modified · honest by design
