🛡️ Sentinel Ensemble — How the Investigation Runs

One autonomous run, top to bottom · Step 0 → 16 · the AI (Qwen on Alibaba Cloud DashScope) is called 5 times (✦), code checks it at every turn
⚙ CONDUCTOR · run_pipeline.py · deterministic Python
owns every step and is the only memory — the AI acts only inside the ✦ AI boxes, and never sees its own previous output
🧠 INFERENCE · Qwen models on Alibaba Cloud (DashScope)
the 5 ✦ AI calls run on Qwen — qwen3.7-max (keystone) · qwen-plus (volume) — over the provider seam src/sift_sentinel/llm_provider.py (chosen by env; Anthropic supported as a zero-regression fallback). Evidence and tools stay local; only prompts/responses cross to Qwen Cloud.

10
🧪 Validator — code checks the AI
every claim is checked against the real evidence → MATCH · MISMATCH · UNRESOLVED. A mismatch blocks the finding (not dropped — sent to Step 13AA).
▼
11
AI CALL 3 — ReAct investigation ✦ AI
the AI re-investigates its own suspicious findings with live tools and can overturn its own verdict.
⭐ SELF-CORRECTION #1  "I flagged this — I tested it — it's actually safe." A scary alert becomes Assessed benign.
▼
12
♻️ Claim rescue (deterministic — no AI)
if a finding's link to its evidence broke, code re-attaches it instead of losing the finding.
▼
13
📊 Confidence calibration (code, not feelings)
3+ independent evidence types = HIGH · 2 = MEDIUM · 1 = LOW; seen in disk AND memory = upgraded.
⭐ SELF-CORRECTION #2 · Step 13AA ⟦ AI ✦ ⟧ — final re-judge  every still-uncertain finding is judged one last time — code gates promotion: no evidence, no confirm. Can rescue a missed threat, never re-litigates a settled one.
▼
14
AI CALL 4 — Report ✦ AI
writes the human report from the validated truth buckets — the prose can never contradict them.
▼
15
🔒 Fingerprint #2 — must equal #1
the "after" fingerprint is compared to the "before." Identical = evidence never modified, proven by math.

📋 Delivered
a clear report — verdict, attack timeline, who did what, remediation — plus findings table, network IOCs, and a full audit trail.
